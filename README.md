思路：  我们的想法是输入关键词，进行抓取
1.首先创建一个抓取的类，进行基础的设置
2.然后进行浏览器生命周期管理，具体的就是创建页面，重启页面，安全打开页面，打印日志等
3.接着就是打开亚马逊网址并登录，由于考虑到不清理资源会导致下次抓取出现问题这个情况，我们在之前配置的是无痕浏览
4.然后，输入搜索词并获得搜索结果页面
5.获得搜索结果中产品的asin，因为在同一个网站中，一个商品对应唯一一个asin，网址都肯定含有asin
6.判断是否有下一页的搜索结果，这个没有写好，我的总是显示没有，希望大家可以帮我改一下
7.将获得的asin拼接成产品链接
8.进入这个产品链接，获得“查看全部评论”的链接，就是“see more reviews”
9.将asin与评论进行对应，我不知道这个有什么用，但是ai建议我弄
10.进入到评论页里面，点击下一页。这个函数的作用就是先抓取评论的那一块，翻页
11.判断是否有下一页评论
12.从评论页面提取评论数据，主要是提取评分，评论标题，评论内容，时间，地点，忘记提取款式了，呜呜呜
时间和地点本来是连在一起的，我们把它分开了
13.统一时间格式
14.文本规范化处理
15.按顺序尝试多个CSS选择器提取评论文本，适配不同页面结构，保证爬虫稳定运行
这个函数刚开始是我自己写的，但是不知道为什么我写的不能翻页，代码告诉我已经翻页了，浏览器也翻页了，但是抓取的还是第一页的内容
16.保存
